# Chrome Web Store 게시 - 필수 항목 작성

## 📋 개인 정보 보호 관행 탭

### 1️⃣ 단일 목적 설명

```
이 확장 프로그램은 웹쇼핑 시 상품의 최저가를 자동으로 찾아주는 
가격 비교 도구입니다. 사용자가 온라인 쇼핑몰에서 상품을 볼 때,
다나와, 11번가, 쿠팡 등의 가격 비교 사이트에서 최저 판매가와 
링크를 실시간으로 검색하여 확장 프로그램 팝업으로 표시합니다.

기능:
- 상품명 자동 인식
- 실시간 최저가 검색
- 가격 비교 결과 표시
- 검색 이력 및 통계 제공
```

---

### 2️⃣ 원격 코드 사용을 요청하는 이유

```
✅ 원격 코드 실행 사용 안 함

이 확장은 로컬 JavaScript와 manifest.json에서만 코드를 실행합니다.
eval(), new Function(), executeScript() 등 동적 코드 실행을 사용하지 않습니다.

대신 다음의 안전한 방식만 사용합니다:
- Content Script (정적 파일 기반)
- Service Worker 통신
- DOM 조작 (안전한 텍스트 설정)
- API 호출 (HTTPS만)
```

---

### 3️⃣ 호스트 권한 사용을 요청하는 이유

```
요청 호스트: https://search.danawa.com/*, https://prod.danawa.com/*

이유:
- 검색 페이지(https://search.danawa.com/dsearch.php)에서 
  상품 정보(가격, 상품명, 링크) 추출
  
- 상품 상세 페이지(https://prod.danawa.com/info/)에서 
  최저가 및 판매처 정보 수집

데이터 수집 범위:
- 상품명, 가격, 이미지 URL, 구매 링크만 수집
- 개인정보(로그인 정보, 쿠키, 세션) 수집 안 함
- 수집된 데이터는 로컬에만 저장, 제3자에 미공개
```

---

### 4️⃣ activeTab 권한을 요청하는 이유

```
현재 활성 탭에서 상품명/텍스트를 추출하기 위해 필요합니다.

용도:
- 사용자가 확장 아이콘을 클릭했을 때 현재 보고 있는 
  웹페이지의 상품 정보를 자동으로 인식
  
- 페이지의 <title>, 메타 정보에서 상품명 추출
- 선택된 텍스트가 있으면 그것을 상품명으로 사용

제한사항:
- 사용자가 명시적으로 확장 아이콘을 클릭할 때만 작동
- 백그라운드에서 자동으로 탭 정보 수집 안 함
```

---

### 5️⃣ storage 권한을 요청하는 이유

```
사용자의 검색 이력, 캐시, 설정을 로컬에 저장하기 위해 필요합니다.

저장되는 데이터:
1. 검색 이력
   - 검색한 상품명, 검색 시간, 결과 (최저가, 링크)
   - 사용 목적: 인기 상품 통계, 빠른 재검색

2. 캐시 데이터
   - 검색 결과 캐시 (6시간 유효)
   - 사용 목적: 반복 검색 시 빠른 응답

3. 사용자 설정
   - 알림 여부, 검색 대상 사이트 선택
   - 사용 목적: 사용자 맞춤형 경험 제공

데이터 보안:
- 모든 데이터는 chrome.storage.local에만 저장 (클라이언트)
- 외부 서버 업로드 안 함
- 브라우저 삭제 시 함께 삭제
```

---

### 6️⃣ offscreen 권한을 요청하는 이유

```
백그라운드에서 HTML 렌더링이 필요한 경우 사용합니다.

용도:
- Service Worker에서 직접 DOM 조작이 불가능하므로,
  필요시 HTML을 오프스크린 문서에 로드하여 파싱
  
- 예: HTML 응답에서 가격 정보 추출 시 DOM API 활용

실제 사용 최소화:
- 대부분은 정규표현식으로 HTML 파싱 (효율적)
- offscreen은 필요한 경우만 선택적 사용
```

---

### 7️⃣ scripting 권한을 요청하는 이유

```
Content Script를 주입하여 웹페이지의 DOM에 접근하기 위해 필요합니다.

용도:
- 활성 탭의 HTML 파싱 및 데이터 추출
- 확장 팝업 UI를 페이지에 표시
- 사용자 상호작용 감지 (클릭, 입력)

제한사항:
- 입력 필드, 로그인 정보 접근 안 함
- 민감한 정보(개인정보, 결제 정보) 수집 안 함
- CSS 클래스, 구조 변경 안 함 (페이지 기능 유지)
```

## 📸 스크린샷/동영상 (필수: 1개 이상)

### 추천 스크린샷 항목

1. **팝업 UI - 검색 결과 표시**
   - 상품명 입력칸
   - 최저가 결과 표시
   - 가격 비교 차트
   
2. **옵션 페이지 - 설정**
   - 검색 대상 사이트 선택
   - 알림 설정
   
3. **통계 페이지**
   - 검색 이력
   - 인기 상품 순위

### 동영상 시나리오 (15-30초)
```
1. 온라인 쇼핑몰에서 상품 검색
2. 확장 아이콘 클릭
3. 자동으로 상품명 인식
4. 최저가 검색 결과 표시
5. "링크 보기" 버튼 클릭으로 구매 페이지 이동
```

## 🔐 추가 보안 정책

### 개인정보 미수집 항목

```
이 확장은 다음을 수집하지 않습니다:

❌ 로그인 정보 (아이디, 비밀번호)
❌ 결제 정보 (카드 번호, 계좌번호)
❌ 쿠키 또는 세션 토큰
❌ 브라우징 히스토리 (명시적 검색만 기록)
❌ IP 주소, MAC 주소
❌ 위치 정보
❌ 카메라, 마이크
```

### 외부 연결

```

- search.danawa.com (검색 페이지 크롤링)
- prod.danawa.com (상품 상세 크롤링)
- 기타 가격 비교 사이트

모든 연결:
✅ HTTPS/보안 채널 사용
✅ 필요한 정보만 요청 (상품명, 가격)
✅ 개인정보 미전송
```
